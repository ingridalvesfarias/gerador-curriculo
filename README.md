# 📄 Gerador de Currículo Profissional - RealTime (Versão Mobile & Multi-Page)

Este projeto é um gerador de currículos dinâmico que permite a criação de documentos profissionais com visualização em tempo real. Desenvolvido com foco em *minimalismo*, *acessibilidade móvel* e *perfeição na impressão A4*.

## 🚀 Funcionalidades Atualizadas
* *Edição em Tempo Real:* Sincronização instantânea entre o editor e a folha de visualização.
* *Responsividade Total:* Interface adaptada para smartphones, tablets e notebooks.
* *Suporte a Múltiplas Páginas:* O conteúdo flui automaticamente para novas folhas conforme o volume de informações aumenta, sem cortes.
* *Impressão Blindada:* CSS otimizado para gerar PDFs limpos, removendo elementos da interface e cabeçalhos desnecessários do navegador.
* *Valores Padrão Dinâmicos:* Garante que o layout nunca "quebre" ou fique vazio, mantendo o design sempre profissional.

## 🛠️ Tecnologias
* *HTML5:* Estrutura semântica.
* *CSS3:* Flexbox, Media Queries (Mobile First) e @media print.
* *JavaScript (ES6+):* Manipulação de DOM para atualização em tempo real e lógica de listas automáticas.

## 📱 Visualização Mobile
Em dispositivos móveis, o editor é posicionado no topo da tela para facilitar a entrada de dados, enquanto o currículo é exibido logo abaixo em largura total, garantindo uma experiência de uso fluida.
